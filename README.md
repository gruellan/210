# 210

To run:
- Open in Intellij
- Go to File > Project Structure > Modules
- Select Java 11 as the Module SDK
- Press Alt+Insert, then press 1
- Select the jsfml.jar file and click OK
- Open the Intellij terminal and type: javac -cp jsfml.jar;. *.java && java  -cp jsfml.jar;. Main


Now the game should be running! :)
